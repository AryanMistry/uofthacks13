# 3D Models Directory

Place your 3D furniture models (.glb or .gltf files) in this directory.

Example models you might want to add:
- sofa.glb
- coffee-table.glb
- lamp.glb
- bookshelf.glb
- rug.glb

You can find free 3D models at:
- https://sketchfab.com
- https://polyhaven.com
- https://free3d.com

Make sure models are optimized for web use (compressed, low poly count).
